## Kullanılan Paketler

### Backend
- `firebase_core: ^3.13.0` - Firebase başlatma
- `firebase_auth: ^5.5.2` - Kullanıcı kimlik doğrulama
- `cloud_firestore: ^5.6.6` - Gerçek zamanlı veritabanı
- `firebase_storage: ^12.4.5` - Dosya depolama
- `shared_preferences: ^2.5.3` - Yerel depolama

### Frontend
- `intl: ^0.20.2` - Tarih/saat formatlama
- `url_launcher: ^6.3.1` - Belge linklerini tarayıcıda açma
- `file_picker: ^8.1.7` - Dosya seçici

### Flutter Widget'ları
- `StreamBuilder` - Gerçek zamanlı veri dinleme
- `ListView.builder` - Dinamik liste
- `RefreshIndicator` - Pull to refresh
- `SegmentedButton` - Rol seçimi
- `FilterChip` - Başvuru filtreleme
- `ExpansionTile` - Öğrenci listesi açılır kart
- `SnackBar` - Bildirim banner
